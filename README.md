
## 🎮 Особливості

- 🌈 Градієнтне тіло змійки з плавним “гойданням”  
- 💀 Крутий Game Over екран із можливістю рестарту (клавіша **R**)  
- ⚡ Прискорення змійки при з’їданні яблука  
- 🕹 Стартове меню (SPACE → Play, ESC → Exit)  
- 🔁 Плавний рух і поворот голови  
- 🎨 Стильний темний фон та сітка для кращої орієнтації  

---

## 🖥 Як грати

1. Запусти `snake.py` у Python 3.x з встановленим `pygame`.  
2. Використовуй стрілки ↑ ↓ ← → для руху змійки.  
3. Їж яблука (червоні квадрати), щоб набирати очки.  
4. Не врізайся в стіни і в себе.  
5. Після смерті натисни **R** для рестарту гри.
